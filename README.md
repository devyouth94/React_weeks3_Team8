## 🧭 File Directory

```bash
/src
  ├── components //컴포넌트 폴더
  │   ├── common //여러 페이지에 공통으로 들어가는 컴포넌트 관리
  │   ├── elements //전역에 사용할 엘리먼트 관리
  │   └── feature //각 기능별 컴포넌트 관리
  │
  ├── pages //페이지 관리
  │
  ├── redux //리덕스 관리
  │   ├── config //스토어 관리
  │   └── slices //슬라이스 관리
  │
  ├── shared //라우터 관리
  │
  ├── App.js
  └── index.js

```
